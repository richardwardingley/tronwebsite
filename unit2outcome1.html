1.7 Describe which are the appropriate filetypes for websites

There are many file types that can be used for websites, these include:

.AVI — Audio/Video Interleaved, a standard video format that doesn’t stream
.CSS — Cascading StyleSheet, to link with and style HTML pages
.DOC — Microsoft Word DOCument
.EXE — EXEcutable file, used when installing programmes
.GIF — Graphics Interchange Format, a brief moving image
.HTML/ .HTM — HyperText Markup Language file, to make website pages
.JPG/ .JPEG — Joint Photographic Experts Group file, an image format
.MIDI/ .MID — Musical Instrument Digital Interface, keyboard music files
.MP3 — MPEG Layer 3 sound file, highly compressed sound files
.MPEG/ .MPG — Motion Picture Experts Group file, movie streaming
.MOV/ .QT — QuickTime MOVie, movie streaming
.PDF — Portable Document Format, documents look the same on all platforms
.PNG — Portable Network Graphics, pictures
.RAM — Real Audio Movie, streaming video
.RAR — RAR archive, group of slightly compressed mixed files
.TIFF — Tagged Image File Format, can’t be viewed on a browser but download to an image editor
.TXT — TeXT file, basic text file, eg. for notepad
.WAV — WAVe sound file, short not very-compressed sound files
.ZIP — ZIPped file, group of slightly compressed mixed files
